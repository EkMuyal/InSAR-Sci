# InSAR-Sci

Python/Jupyter tools for post-processing, inspection, and scientific analysis of exported GAMMA-derived Sentinel-1 InSAR GeoTIFF products.

This repository is intended for reproducible research workflows around:

- GeoTIFF inventory and metadata inspection
- Pair-manifest validation
- LOS displacement and coherence diagnostics
- Candidate slow slip event screening
- Future atmospheric correction and deformation-analysis tools

Large raster products, GAMMA binaries, outputs, figures, and logs are intentionally excluded from Git.
